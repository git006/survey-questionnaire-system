问卷调查系统--项目总结提示词
系统概述
员工满意度问卷调查系统，包含用户答题端和管理后台。
一、用户答题端
登录要求
- 输入昵称和所在城市两项信息后才能进入问卷页面
- 登录后顶部显示"昵称 (城市)"格式
问卷功能
- 7个预设问题：公司制度、上下班时间、通勤便利、岗位满意度、同事相处、薪资水平、个人成长
- 5分制评分：1分(不满意)到5分(非常满意)
- 选择1分或5分时弹出详细反馈输入框
- 提交后显示成功页面
成功页面
- 标题："提交成功！感谢您的参与，您的反馈对我们非常重要！"
- 三个按钮：
  - "查看我的问卷" - 查看提交结果，底部有"← 返回"按钮(右下角，紫色渐变)
  - "填写新问卷" - 重新填写
  - "退出" - 跳转到登录页面
---
二、管理后台
登录
- 账号：admin，密码：admin
- 右上角显示 admin 账号
首页(dashboard)
- 三个卡片(顺序：问卷设置 → 问题管理 → 问卷反馈结果)
- 问卷反馈结果显示"已收到问卷数(数字)"
问题管理
- 可添加、编辑、删除问题
- 拖拽排序
问卷设置
- 可自定义：问卷标题、副标题、角标提示、建议反馈标题
问卷反馈结果
- 表格列：ID、昵称、各问题评分(带颜色标识)、提交时间、操作
- 每个问题列有下拉筛选(1-5分)
- 右上角"问卷报告图"按钮
统计饼图
- 每个问题一个饼图，5个区块
- 区块面积 = 选择该分数的人数
- 区块颜色：绿色=1、蓝色=2、红色=3、黄色=4、灰色=5
- 区块中心显示对应分数数字(不是人数)
- 鼠标悬停不显示信息
---
三、数据库表
- users: id, nickname, city, created_at
- questions: id, title, order_num, created_at
- surveys: id, user_id, answers(JSON), suggestions, created_at
- settings: id, title, subtitle, badge, suggestion_title
